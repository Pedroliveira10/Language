# Roadmap

## 1. Stabilize

- Preserve the current app in `public/index.html`.
- Add automated content and behavior tests.
- Document content conventions and supported locales.

## 2. Extract

- Move shared storage, audio, progress, and navigation logic into `src/services`
  and `src/app`.
- Replace duplicated UI with components.
- Move embedded lesson data into language curriculum files.

## 3. Expand

- Add complete A1 curricula and spaced-review scheduling.
- Add recorded native-speaker audio with text-to-speech fallback.
- Add offline caching, import/export, accessibility checks, and analytics with
  explicit consent.

## 4. Scale

- Add A2 and higher content.
- Add optional accounts and cross-device sync.
- Add a content-authoring and validation workflow.
