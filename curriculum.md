# Content guidelines

1. Use UTF-8 and preserve diacritics.
2. Keep one learning objective per exercise.
3. Use natural, contemporary language and identify formal or informal register.
4. Accept legitimate spelling or phrasing variants when the task allows them.
5. Include grammatical gender and plural forms where relevant.
6. Keep Dutch, Polish, and European Portuguese locale metadata explicit.
7. Do not add audio without a transcript and locale.
8. Avoid stereotypes, sensitive personal data, and culturally misleading examples.
9. Assign stable IDs; published IDs must not be reused for different content.
10. Run `npm test` and `npm run check:content` before publishing.
